This plugin transforms ES2015 modules to CommonJS.

## Installation

```sh
$ npm install babel-plugin-transform-es2015-modules-commonjs
```

## Usage

Add the following line to your `.babelrc` file:

```json
{
  "plugins": ["transform-es2015-modules-commonjs"]
}
```
