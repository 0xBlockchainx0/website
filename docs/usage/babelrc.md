The entire range of Babel API [options](/docs/usage/options) are allowed.

**Example:**

```json
{
  "stage": 1,
  "loose": ["es6.modules", "es6.classes"],
  "ignore": [
    "foo.js",
    "bar/**/*.js"
  ]
}
```
